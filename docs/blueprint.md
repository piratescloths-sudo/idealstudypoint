# **App Name**: EduFlow

## Core Features:

- Public Facing Content Display: Dynamically display website content such as Home hero section, Featured courses, About us, Upcoming events, Testimonials on dedicated Home, Courses, Admission, Events, and Contact pages, with all data managed through the Admin Dashboard.
- User Inquiry Forms: Capture user submissions via Admission and Contact forms, storing all details securely in Firebase Firestore for administrator review.
- Admin Authentication: Securely authenticate administrative users using Firebase Authentication to restrict access to the Admin Dashboard and its content management features.
- Course Management: Enable administrators to create, edit, delete, and view course details, including uploading course images to Firebase Storage.
- Event Management: Provide administrators with tools to add, modify, and remove event listings, along with uploading associated event images to Firebase Storage.
- CMS for Static Pages & Settings: Allow administrators to update core website content such as homepage sections (hero text, about, testimonials) and overall site settings (contact information, social links) through a dedicated interface.
- AI-Powered Content Tool: Offer an AI tool within the Admin Dashboard to assist in generating engaging course descriptions and compelling event summaries for the public website.

## Style Guidelines:

- The primary color is a balanced, academic blue (#3980AC), chosen to convey trust and knowledge without being overly bold. It contrasts effectively with light backgrounds, promoting readability.
- The background color is a very light, almost white, blue (#EEF4F6), offering a clean and calm canvas that visually harmonizes with the primary blue hue.
- A fresh, energetic turquoise (#4ADBDD) serves as the accent color, analogous to the primary. Its vibrant contrast is ideal for drawing attention to calls-to-action and key elements, adding a modern touch.
- Headlines and prominent text will use 'Poppins', a geometric sans-serif font known for its contemporary and precise appearance. Body text and longer content will utilize 'Inter', a grotesque-style sans-serif, ensuring optimal readability and a modern, objective feel.
- Employ a consistent set of clean, modern, and educational-themed vector icons. Prioritize line-based or subtly filled icons for a sleek, professional look that complements the overall aesthetic.
- Implement a fully responsive design featuring rounded cards with soft shadows. The public website will have a clean, intuitive navigation with a sticky header and mobile menu, while the Admin Dashboard will feature a clear sidebar layout for efficient content management.
- Incorporate subtle hover animations on interactive elements, smooth transitions between pages, and micro-interactions for an enhanced user experience. Utilize loading skeletons during data fetches and implement toast notifications for user feedback on actions.
